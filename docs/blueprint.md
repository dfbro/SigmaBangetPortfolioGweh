# **App Name**: CipherSynth Portfolio

## Core Features:

- CTF Write-Up Repository: Dedicated section to display detailed write-ups of Capture The Flag challenges, categorized for easy navigation.
- Technical Project Showcase: Showcase technical projects with descriptions, relevant links (e.g., GitHub), and demo screenshots or videos.
- Achievement and Certification Gallery: A visually engaging gallery to highlight cybersecurity certifications, badges, and professional achievements.
- Interactive Professional 'About Me' Page: A dynamic and professional section detailing the user's career journey, skills, and interests in cybersecurity.
- AI-Powered Bio & Content Assistant Tool: An AI tool that assists the user in drafting or refining professional summaries for their 'About Me' page or succinct descriptions for projects and write-ups, ensuring consistent tone and keyword optimization.

## Style Guidelines:

- Dark scheme for a modern 'hacker' aesthetic. Primary interactive color: vibrant neon green (#57F906), signifying active terminal commands and key actions.
- Background color: A very dark, subtle green-tinged black (#131711), providing depth and reducing eye strain in a dark theme.
- Accent color: A bright, electric cyan (#33EBCA), analogous to the primary green, used for secondary highlights and data visualization to complement the technical theme.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, techy feel. Body font: 'Inter' (sans-serif) for excellent readability on technical content.
- Code blocks font: 'Source Code Pro' (monospace) for displaying code snippets and terminal outputs clearly within write-ups.
- Use sharp, minimalist, line-art icons. Geometric shapes and subtle digital motifs will reinforce the cybersecurity theme. Deep blue tones can be incorporated into these graphical elements for accents.
- A clean, modular layout with strong contrasts to enhance readability. Information is organized in clear cards or panels against the dark background. Subtle, dark blue panels or section backgrounds can be used for informational grouping to fulfill the requested dark blue aesthetic.
- Subtle animations mimicking terminal output (e.g., text typing, blinking cursor), hover effects with neon glows, and transitions that feel snappy and digital to create an immersive 'hacker' experience.
